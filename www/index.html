⚒ ForgeOS

A Terminal Operating Environment Built in C

v1.0.0 "Ironclad"

This page is served by the ForgeOS HTTP Server — a multithreaded C web server with a thread-pool, static file hosting, and MIME detection.

🐚 Shell

  • Command parsing & tokenization
  • Pipes, redirection, &&/||
  • Background processes (&)
  • 500-entry history + ↑↓ nav
  • Tab completion
  • Built-ins: cd, echo, export, jobs

📝 Editor

  • Open / save files
  • Cursor & viewport movement
  • Insert, delete, newline
  • Incremental search (Ctrl-F)
  • Syntax highlighting: C, Python, Shell
  • Dirty-file tracking (Ctrl-S to save)

🌐 HTTP Server

  • 8-thread pool, keep-alive
  • Static file serving
  • Directory listing
  • MIME type detection (20+ types)
  • GET / HEAD support
  • Request stats

💾 Allocator

  • 16 MB static heap
  • Free-list with coalescing
  • forge_malloc / free / calloc
  • Thread-safe (mutex-protected)
  • Integrity checker
  • Allocation statistics

📊 Process Monitor

  • Reads /proc filesystem
  • Live CPU % per process
  • Memory usage bars
  • Sort by CPU / MEM / PID
  • Keyboard-driven TUI
  • System load + uptime

🔌 Plugin System

  • dlopen() shared libraries
  • ABI versioning
  • Register shell commands
  • Startup/shutdown hooks
  • Example: hello_plugin.so
  • Hot-load from plugins/

Quick Start

$ git clone https://github.com/yourusername/forgeos
$ cd forgeos && make
$ make run         # interactive shell
$ ./bin/forgeos -e myfile.c    # open editor
$ ./bin/forgeos -w 8080 ./www  # this server!
$ ./bin/forgeos -p             # process monitor
$ make test        # run test suite
